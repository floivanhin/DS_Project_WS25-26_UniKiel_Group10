# Bundesliga Weather Analysis Dashboard

This project is a small **Vue 3 dashboard** that analyzes how weather conditions during Bundesliga matches influence goal scoring.

---

# Technologies Used

- Vue 3
- TypeScript
- Vite
- Vue Router
- Plotly.js

---

# Prerequisites

Before running the project you need to install:

- **Node.js**
- **npm**

You can check if they are installed with:

```bash
node -v
npm -v

npm install

npm install plotly.js-dist-min

npm install vue-router

npm run dev
```
